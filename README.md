# asd-sync

A tool for rendering the ALPS application diagram in real time.

## Requirement

 * npm
 * asd

## Installation

```
npm install
```

## Start ASD web service


```
npm start
```

The default path os alps/alps.json

```
npm start -- --profile foo/alps.json
```

You can specify profile path as as described above.
